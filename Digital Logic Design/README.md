# Digital Logic Design

Hardware-implemented combinational and sequential logic circuits using 74xx-series TTL ICs on breadboard. All circuits were physically constructed and verified against experimental truth tables.

## Labs

| # | Lab | Type | Key IC(s) |
|---|-----|------|-----------|
| 01 | [De Morgan's Theorem Verification](01_demorgan_theorem_verification/) | Combinational | 7400, 7402, 7404, 7408, 7432 |
| 02 | [Universal Gate Implementation](02_universal_gate_implementation/) | Combinational | 7400, 7402 |
| 03 | [Binary Adder/Subtractor](03_binary_adder_subtractor/) | Arithmetic | 7483, 7486 |
| 04 | [Magnitude Comparator](04_magnitude_comparator/) | Arithmetic | 7485, 7486 |
| 05 | [Encoder & Decoder Circuits](05_encoder_decoder_circuits/) | Data | 7400, 7432 |
| 06 | [Multiplexer & Demultiplexer](06_multiplexer_demultiplexer/) | Data Routing | 7411, 7432, 7404 |
| 07 | [RS & D Flip-Flop](07_rs_d_flipflop/) | Sequential | 7410 |
| 08 | [JK & T Flip-Flop](08_jk_t_flipflop/) | Sequential | 7402, 74011 |
